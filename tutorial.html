基本教程

1.基本命令

  • hexo n “我的博客” == hexo new “我的博客” #新建文章
  • hexo p == hexo publish
  • hexo g == hexo generate#生成
  • hexo s == hexo server #启动服务预览

  • hexo d == hexo deploy#部署

  • 完成后部署
    hexo deploy -g

2.图片引用

  • 在hexo/source目录下新建一个img文件夹，将图片放入该文件夹下，插入图片时链接即为/img/图片名称
